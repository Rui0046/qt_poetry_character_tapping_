# qt_poetry_character_tapping_
项目简介：南开大学25级工科大类C++大作业，点字成诗小游戏，通过在九宫格和十二宫格中识别正确的诗句积分闯关

##v1.0初始代码

###已实现功能：
1.基础界面搭建（九宫格、十二宫格布局）
2.诗句数据读取和基础显示
3.点击交互逻辑

###已知问题：
1.主页显示按键不全，游戏大标题缺失
2.界面适配问题：背景图未能成功插入
3.分数设置逻辑有误：本应是选对一整句诗得一分，现有代码是选出答案中含有的正确字即得分
4.用户体验不足，缺少画面和音效
